# 盲派命理金口诀 (bí truyền) — 断运篇 + 十神专论

Source: 豆瓣 note 399289555 (盲派金口诀). Day la bo CAU QUYET bí truyền cua manh phai (vi tri tru × thap than → ket qua).
## 断运篇 (key): 长生遇杀最有灾 / 七杀逢财不可当 / 劫怕见财财怕劫 / 财官切忌入空亡 / 一见伤官有一伤 / 金衰必在火上死火死必定水上亡...
## 十神专论: mỗi thần (正财/偏财/正官/七杀/正印/偏印/食神/伤官/比劫) × 年上/月上/日上/时上 → 1 cau quyết đoán.
